# 用户个人中心

## 功能概述

用户个人中心是一个完整的用户信息管理界面，采用左侧摘要卡片 + 右侧 Tab 分区的布局方式，提供了全面的用户信息管理和个性化设置功能。

## 页面结构

```
profile/
├── index.vue                          # 主页面（布局容器）
└── components/
    ├── ProfileSummary.vue             # 左侧摘要卡片
    ├── BasicInfo.vue                  # 基础信息
    ├── SecuritySettings.vue           # 账号安全
    ├── NotificationSettings.vue       # 通知设置
    ├── OperationLogs.vue              # 操作日志
    └── PreferencesSettings.vue        # 偏好设置
```

## 功能模块

### 1. 左侧摘要卡片 (ProfileSummary)

**功能：**
- 显示用户头像、昵称、用户名
- 显示账号状态（正常/锁定/过期）
- 显示基本信息（性别、部门、手机号、邮箱、注册时间）
- 显示安全等级（根据绑定情况计算）
- 显示绑定状态（手机、邮箱）

**特点：**
- 手机号和邮箱自动脱敏显示
- 安全等级动态计算（绑定越多，等级越高）
- 桌面端 sticky 定位，移动端自适应

### 2. 基础信息 (BasicInfo)

**功能：**
- 查看和编辑用户基本信息（昵称、性别）
- 上传和更换头像
- 查看只读信息（用户名、手机号、邮箱、部门、注册时间）

**交互：**
- 点击"编辑资料"按钮进入编辑模式
- 支持头像上传（JPG、PNG，最大 2MB）
- 表单验证和错误提示
- 保存成功后自动刷新用户信息

### 3. 账号安全 (SecuritySettings)

**功能：**
- 修改密码（需要输入原密码）
- 绑定/更换手机号（需要验证码和原密码）
- 绑定/更换邮箱（需要验证码和原密码）
- 登录设备管理（查看最近登录设备）
- 三方账号绑定/解绑

**安全措施：**
- 密码使用 RSA 加密传输
- 敏感操作需要二次确认
- 修改密码后强制重新登录
- 显示登录设备详情（浏览器、操作系统、IP、地点）

### 4. 通知设置 (NotificationSettings)

**功能：**
- 通知渠道开关（站内信、邮件、短信）
- 通知类型选择（系统消息、任务提醒、安全告警、操作日志）
- 告警级别阈值设置（信息、警告、错误、严重）

**特点：**
- 实时保存设置
- 清晰的分组和说明

### 5. 操作日志 (OperationLogs)

**功能：**
- 查看个人操作日志（登录记录）
- 筛选功能（状态、时间范围）
- 分页展示
- 导出日志

**显示信息：**
- 操作类型（登录/退出）
- 操作时间
- IP 地址
- 登录地点
- 设备信息（浏览器/操作系统）
- 操作结果

### 6. 偏好设置 (PreferencesSettings)

**功能：**
- 语言设置（中文/英文）
- 主题模式（浅色/深色/跟随系统）
- 主题色选择（预设 7 种颜色）
- 布局模式（侧边栏/顶部栏/混合）
- 内容宽度（固定/流式）
- 紧凑模式开关
- 显示设置（面包屑、标签页、页脚）
- 默认首页路径设置
- 恢复默认设置

**特点：**
- 实时预览效果
- 一键恢复默认
- 设置持久化保存

## 响应式设计

### 实现方式
使用 CSS Grid 布局实现真正的响应式设计，不依赖组件库的响应式 API。

### 断点说明

#### 移动端（<768px）
- **布局**：单列垂直布局
- **摘要卡片**：全宽显示，取消 sticky 定位
- **内容区域**：全宽显示
- **间距**：16px
- **Padding**：16px

#### 平板端（768px ~ 1023px）
- **布局**：单列垂直布局
- **摘要卡片**：全宽显示
- **内容区域**：全宽显示
- **间距**：20px
- **Padding**：20px

#### 桌面端（1024px ~ 1279px）
- **布局**：左右两列布局
- **摘要卡片**：固定宽度 320px，sticky 定位
- **内容区域**：自适应剩余宽度
- **间距**：24px
- **Padding**：24px

#### 大屏（1280px ~ 1535px）
- **布局**：左右两列布局
- **摘要卡片**：固定宽度 360px
- **内容区域**：自适应剩余宽度
- **间距**：24px
- **Padding**：24px

#### 超大屏（≥1536px）
- **布局**：左右两列布局
- **摘要卡片**：固定宽度 380px
- **内容区域**：自适应剩余宽度
- **间距**：24px
- **Padding**：24px
- **最大宽度**：1600px（居中显示）

### CSS Grid 配置

```scss
.profile-layout {
  display: grid;
  grid-template-columns: 1fr;  // 移动端：单列
  gap: 16px;
  max-width: 1600px;
  margin: 0 auto;
}

// 桌面端及以上：左右布局
@media (min-width: 1024px) {
  .profile-layout {
    grid-template-columns: 320px 1fr;  // 左侧固定，右侧自适应
    gap: 24px;
  }
}
```

## API 接口

### 用户资料相关
- `uploadAvatar` - 上传头像
- `updateUserBaseInfo` - 修改基本信息
- `updateUserPassword` - 修改密码
- `updateUserPhone` - 修改手机号
- `updateUserEmail` - 修改邮箱
- `listUserSocial` - 获取三方账号列表
- `bindSocialAccount` - 绑定三方账号
- `unbindSocialAccount` - 解绑三方账号

### 日志相关
- `authApi.listLoginLog` - 获取登录日志列表
- `authApi.exportLoginLog` - 导出登录日志

## 安全与合规

### 数据脱敏
- 手机号：显示为 `138****5678`
- 邮箱：显示为 `ab***@example.com`

### 密码安全
- 密码复杂度要求（8-20 位，包含大小写字母、数字和特殊字符）
- 密码 RSA 加密传输
- 修改密码后强制重新登录

### 操作留痕
- 所有敏感操作记录日志
- 支持日志导出和审计

### 二次确认
- 修改密码需要输入原密码
- 解绑账号需要二次确认
- 退出所有设备需要确认
- 恢复默认设置需要确认

## 注意事项

1. **权限控制**：确保用户只能修改自己的信息
2. **数据验证**：前后端都需要进行数据验证
3. **错误处理**：提供友好的错误提示
4. **加载状态**：异步操作显示加载状态
5. **成功反馈**：操作成功后给予明确反馈
6. **数据刷新**：修改后及时刷新用户信息
